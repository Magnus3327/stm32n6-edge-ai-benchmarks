# NPU Performance Benchmark & Analysis

This repository is part of a curricular internship project at the **University of Trento**, carried out within the **DISI** (Department of Information Engineering and Computer Science) under the supervision of professors **Vella** and **Yildirim**.

## Project Overview

The project focuses on the optimization and performance evaluation of deep learning operators (single-layer microbenchmarks) deployed on the NPU of the **STM32N6** microcontroller.

Specifically, this repository compares:
- **Actual Hardware Measurements (Baseline)** vs. **Static Compiler Predictions (Estimated)**.
- Two compiler generations from STMicroelectronics: **AI Core v2.2** (via STM32CubeMX / X-CUBE-AI) and **AI Core v4.0** (via STM32 AI Studio).
- Two execution platforms: the local physical **STM32N6 Nucleo** board and the **STM32N6 Discovery Kit (DK)** board accessed via **ST DevCloud**.

## Repository Structure

The repository is organized as follows:

- **`Models/`**: Contains the target neural network operator models (quantized/integer formats) used for benchmarking.
- **`Output/`**: Contains raw compiler logs, compilation reports, and execution outputs from local tools and ST DevCloud.
- **`Results/`**: Structured performance data in CSV format:
  - `baseline/`: Actual latencies, memory footprint, and Notes measured on the hardware.
  - `estimated/`: Static RAM and Flash footprint estimations generated by the compilers.
- **`Report/`**: Contains the benchmarking deliverables:
  - `NPU_Performance_Analysis.ipynb`: Jupyter Notebook containing consolidated performance comparison tables and plots.
  - `NPU_Performance_Analysis.html` & `NPU_Performance_Analysis.pdf`: Exported report versions formatted for printing and sharing.
  - `plot/`: Exported plot images (inference latency, RAM memory usage, MACC complexity, and statistical distributions).
- **`generate_nb.py`**: Python script used to programmatically generate the Jupyter Notebook and compile the reports.

## Getting Started

To regenerate the Jupyter Notebook and compile the HTML and PDF reports, follow these steps:

1. **Create and activate a Python virtual environment:**
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. **Install the required dependencies:**
   ```bash
   pip install nbformat nbconvert pandas seaborn matplotlib ipykernel jinja2 ipython
   ```

3. **Generate the empty notebook structure:**
   ```bash
   python3 generate_nb.py
   ```
   This will write the template notebook to `Report/NPU_Performance_Analysis.ipynb`.

4. **Execute and compile the reports:**
   Run the notebook cells to execute the calculations and save the charts directly inside the notebook file:
   ```bash
   jupyter nbconvert --to notebook --execute --inplace Report/NPU_Performance_Analysis.ipynb
   ```
   Export the notebook to HTML format:
   ```bash
   jupyter nbconvert --to html Report/NPU_Performance_Analysis.ipynb
   ```
   Print the HTML report to a high-quality PDF using Google Chrome's headless mode:
   ```bash
   "/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless --no-sandbox --print-to-pdf="Report/NPU_Performance_Analysis.pdf" "Report/NPU_Performance_Analysis.html"
   ```
